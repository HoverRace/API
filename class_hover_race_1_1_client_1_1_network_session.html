Detailed Description

The NetworkSession is a child of the ClientSession; it is a networked game session.

The NetworkSession class itself manages the network interface, connected clients, scores, and general game data.

void HoverRace::Client::NetworkSession::WriteNet ( )
protected

Send all necessary data to clients.

This is called by NetworkSession::Process() as part of the game loop.

If we are the server, we must send clock updates 12 and 8 seconds before the game starts.

If our hovercraft has been created, we must broadcast that. Otherwise, we broadcast its state and statistics.

Then, we check if any clients are disconnected and remove them if necessary.
