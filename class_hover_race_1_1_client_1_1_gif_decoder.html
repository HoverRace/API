bool HoverRace::Client::GifDecoder::Decode ( const unsigned char *  pGifStream,
int  pStreamLen 
)

Load a GIF image.

This will replace the currently-loaded image. Animated GIFs are not yet supported – only the first frame of an animated GIF will be loaded.

Parameters
pGifStreamThe GIF data buffer (may not be NULL).
pStreamLenThe length of the buffer.
Returns
true if the load succeeded, false otherwise (current image will still be cleared).
